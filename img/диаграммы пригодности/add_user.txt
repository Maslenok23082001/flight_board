note as scenario
Главная последовательность:
    1) Администратор в главном окне нажимает кнопку “Администрирование”.
2)Открывается окно администратора.
3)Администратор нажимает кнопку “Управление пользователями”.
4)Открывается окно со списком всех пользователей.
5)Администратор нажимает кнопку “Добавить”.
6)Открывается окно добавления нового пользователя, в котором администратор вносит необходимые данные,
 а так же, выбирает статус пользователя(Диспетчер, пассажир). После этого нажимает кнопку “Сохранить”.
7)Открывается окно управления пользователями.

end note
