actor Администратор

boundary "Главное окно" as main_window
boundary "Управление пользователями" as control_users
boundary "Меню администратора" as menu_admin
 
control "Завершение работы программы" as exit_program
control "Удаление пользователя" as del_user
control "Выйти из меню адиминистратора" as exit_window_admin
control "Завершить работу окна упраления пользователями" as exit_udit_users
Администратор -- main_window 

main_window --> exit_program : "Завершить работу"
main_window --> menu_admin : "Переход в меню администратора"
menu_admin --> exit_window_admin : "Завершить работу окна 'меню администратора'"
menu_admin --> control_users : "Переход к списку всех пользователей"
control_users --> exit_udit_users : "Завершить работу окна"
control_users --> del_user : "Удалить пользователя"
del_user --> control_users : "Пользователь успешно удалён"


note as scenario
Главная последовательность:
    1)Администратор в главном окне нажимает кнопку “Администрирование”.
    2)Открывается окно меню администратора, на этом шаге можно завершить работу этого окна.
    3)Администратор в окне 'меню администратора' нажимает кнопку “Управление пользователями”.
    4)Открывается окно 'Управление пользователями' со списком всех пользователей, 
      на данном этапе можно выйти из этого окна.
    5)Администратор выделяет пользователя одним нажатием по нему и нажимает кнопку “Удалить”.
      После чего пользователь удаляется.

end note
@enduml
