@startuml
actor Пассажир

boundary "Главное окно" as main_window
boundary "Интерфейс фильтрации" as filtr_v
 
control "Завершение работы программы" as exit_program
control "Посимвольный ввод пользователем для нахождения желаемого рейса" as poisk_simv
control "Поиск в списке рейсов схожести с введёнными символами" as poisk
control "Элемент поиска найден" as poisk_ok

Пассажир -- main_window

main_window --> filtr_v : "Запись в поле для фильтрации"
main_window --> exit_program : "Выход из программы"
filtr_v --> poisk_simv : "Ввод символов в поле для фильтрации"
poisk_simv --> poisk : "Поиск системы в списке рейсов"
poisk --> poisk_ok : "Успешно найден(ы) рейс(ы)"
poisk_ok --> main_window : "Изменение списка рейсов"


note as scenario
Главная последовательность:
    1)Пассажир в главном окне просматривает список рейсов.
    2)В текстовое поле для фильтрации пассажир посимвольно вводит текст, по которому он хочет отфильтровать рейсы,
      Фильтрация происходит мгновенно.
    3)После чего на главном окне выводиться отфильтрованный список рейсов, если список пуст, значит желаемого рейса в списке нет.


    
end note
@enduml
